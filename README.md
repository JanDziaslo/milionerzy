# Milionerzy
Prosty quiz napisany w python bez użycia zewnętrznych biblotek.
Dokumentacja projektu znajduje sie w zakładce wiki na github.
## Do zrobienia
- [ ]  Aplikacja w python
	- [ ]  losowanie k pytań ze zbioru n pytań
	- [ ]  Kolejnosc odpowiedzi jest losowa
	- [ ]  Surowe pytania i odpowiedzi znajduja sie w osobnym pliku
	- [ ]  Poprawne odpowiedzi znajduja sie w innym pliku (zeby przypadkiem za latwo nie bylo)
	- [ ]  Wyniki oraz imie i nazwisko gracza w jeszcze innym pliku
	- [ ]  debilo odporne
- [ ] Dokumentacja
	- [ ] znajduje sie w zakladce wiki na github 
	- [ ] wyjasnia jak dziala aplikacja
	- [ ] jak z niej korzystac
	- [ ] jak jest zbudowana 
	- [ ] zawiera tworcow
	
## Treść zadania z moodla
```
Napisz aplikację quiz, która losuje k pytań ze zbioru n pytań. Kolejność odpowiedzi również jest losowa.

Pytania i odpowiedzi znajdują się w jednym pliku, poprawne odpowiedzi w drugim, wyniki oraz imię i nazwisko osoby rozwiązującej zapisywane będzie w  trzecim pliku. 

Quiz ma być odporny na błędy użytkownika. 

Przygotuj dokumentację aplikacji (jak działa, jak z niej korzystać, jak jest zbudowana, wpisać twórców)

Prześlij do oceny cały projekt zwierający zarówno plik py jak i 3 pliki txt spakowane do np. zip, rar, 7z. 
```
